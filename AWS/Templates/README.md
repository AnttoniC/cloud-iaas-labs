# AWS Templates (CloudFormation)

Templates organizados por exercício. Dentro de cada pasta, os arquivos numerados (`01-`, `02-`...) seguem a ordem de evolução do exercício, do rascunho à versão final.

| Pasta | Conteúdo |
|---|---|
| `01-servidor-simples/` | VPC + Security Group + instância EC2 — exercício introdutório |
| `02-servidor-web-ssh/` | Servidor com Security Group liberando acesso Web (80) e SSH (22) |
| `03-wordpress/` | Instância EC2 configurada para receber o deploy de um WordPress |
| `04-vpc-multicamada/` | VPC com subnets, route table, internet gateway e múltiplos servidores (Web, BD, SSH) |
| `05-cluster-autoscaling/` | Exercício mais avançado: Auto Scaling Group, Launch Configuration, Load Balancer, RDS, NAT Gateway |
